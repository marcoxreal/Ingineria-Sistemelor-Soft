# Ingineria-Sistemelor-Soft


Tuneboxd: a Letterboxd-like app, but for music.

Rate, review, recommend albums to friends and strangers.

Functionalities:
  - Mark an album as "Listened To", rate it (out of 5 stars), and leave a review
  - See top albums (by rating from the community); they can be filtered by genre(s) and/or artists
  - Make friends on Tuneboxd!
  - Get automatic album recommendations based on what you like
